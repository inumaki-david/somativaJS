# 🚀 Desenvolvimento de Projetos Interativos Front-End

Este repositório contém uma série de projetos desenvolvidos como parte de um desafio de aprendizagem focado em tecnologias web essenciais (**HTML5, CSS3 e JavaScript**). O objetivo é demonstrar a evolução técnica através de três níveis de dificuldade, focando em funcionalidade, interatividade e design responsivo.

## 📋 Sobre o Projeto

O desafio propõe a implementação de três aplicações distintas, cada uma explorando diferentes lógicas de programação e manipulação do DOM (Document Object Model). 

### 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização moderna, Flexbox/Grid e design responsivo.
* **JavaScript:** Lógica de jogo, manipulação de eventos e temporizadores.

---

## 🕹️ Projetos Implementados

### 1. Nível Fácil: Jogo Genius 🟢🔴🟡🔵
Uma releitura do clássico jogo de memória visual e sonora.
* **Funcionalidade:** O sistema gera uma sequência aleatória de cores que o usuário deve repetir.
* **Destaque Técnico:** Manipulação de arrays, funções de `setTimeout` para controle de luzes e tratamento de eventos de clique.

### 2. Nível Médio: Temporizador ⏱️
Uma ferramenta de gestão de tempo com interface dinâmica.
* **Funcionalidade:** Cronômetro com funções de Iniciar, Pausar e Resetar.
* **Destaque Técnico:** Uso de `setInterval`, formatação de strings (00:00:00) e controle de estados da aplicação.

### 3. Nível Difícil: Calculadora Simples Gráfica 🔢
Calculadora com interface completa para operações matemáticas.
* **Funcionalidade:** Realização de operações básicas com uma interface visual que simula um dispositivo real.
* **Destaque Técnico:** Lógica de precedência, tratamento de erros e refinamento de interface (UI).

---

## ⚖️ Critérios de Qualidade

Para garantir a excelência acadêmica e profissional, os projetos seguem os seguintes critérios:

* **Funcionalidade:** Execução correta de todas as tarefas propostas.
* **Interface (UI):** Design intuitivo, limpo e visualmente agradável.
* **Responsividade:** Interfaces que se adaptam a diferentes tamanhos de tela.
* **Qualidade de Código:** Código estruturado, organizado e com comentários explicativos.
