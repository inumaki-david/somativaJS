# Calculadora Básica

Esse é um projeto simples de uma calculadora funcional feita com HTML, CSS e JavaScript.
## O que ela faz

* Soma, subtrai, multiplica e divide.
* Limpa o visor (C) e apaga o último caractere (DEL).
* Design escuro (dark mode).

## Como as coisas funcionam

* HTML: Define os botões e o campo onde os números aparecem.
* CSS: Organiza tudo num grid bonitinho e coloca uns efeitos de clique.
* JavaScript: Pega o que você digita e usa a função eval() pra resolver a conta de uma vez. Se você tentar fazer uma conta impossível, ela avisa que deu "Erro".
